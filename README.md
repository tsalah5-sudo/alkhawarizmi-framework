# 🚀 إطار عمل الخوارزمي | Al-Khwarizmi Framework

إطار عمل (Framework) خفيف، سريع، وآمن مبني بلغة **PHP النقي (Vanilla PHP)**. تم تصميمه خصيصاً لمساعدة المطورين والمصممين على بناء مشاريع الويب وتطبيقاتها بكفاءة عالية وبساطة متناهية، مستوحى من التنظيم والذكاء العددي لعالم الرياضيات الشهير **الخوارزمي**.

A lightweight, fast, and secure PHP framework built with Vanilla PHP. Designed specifically to help developers and designers build web applications with high efficiency and absolute simplicity.

---

## ✨ المميزات الرئيسية | Key Features

* 🗺️ **Smart Routing (توجيه ذكي):** نظام توجيه مرن وسريع يربط الروابط بالمتحكمات بسلاسة.
* 🏗️ **Clean MVC Architecture:** فصل تام بين منطق البرمجة (Backend) وواجهات المصممين (HTML/CSS).
* 🔋 **Built-in Algorithms (خوارزميات مدمجة):** أدوات جاهزة ومعالجات ذكية لتقسيم الصفحات (Pagination) وتصفية البيانات والترتيب السريع (QuickSort).
* 🔒 **Built-in Security (أمان مدمج):** حماية تلقائية ضد ثغرات XSS وتشفير متقدم لكلمات المرور باستخدام خوارزمية `BCRYPT`.
* 💎 **Lightweight & Portable:** كود نظيف وخفيف بدون تعقيدات زائدة، يسهل نقله ونسخه بين الاستضافات (Hosting).
